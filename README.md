> 패키지 매니저로 설치하는 방법 : (https://nodejs.org/ko/download/package-manager/#enterprise-linux-fedora)

>> sudo yum -y install nodejs
>> node -v
> 릴리즈 파일로 설치하는 방법 : tar.xz 받아서 tar -xvf node-v...x64.tar.xz
